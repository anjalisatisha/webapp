CI/CD Pipeline for Web Application Deployment with Docker

Implemented a CI/CD pipeline using Jenkins to automate the deployment of a web application to Apache Tomcat. The project involved building Docker images, storing them in DockerHub, and configuring Jenkins for continuous integration and deployment. The application was deployed in an AWS-hosted Linux environment.

  • Tech Stack: AWS, Linux, Git, GitHub, Maven, Jenkins, Docker, DockerHub, Apache Tomcat
  • Outcome: Gained hands-on experience in creating CI/CD pipelines, containerizing applications with Docker, managing images with DockerHub, and deploying to Apache Tomcat.
